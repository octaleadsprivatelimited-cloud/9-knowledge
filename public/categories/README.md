# Category images (homepage cards)

Place images here to show as background on category cards. They are used when the category has no image set in the admin panel.

**Naming:** Use the category **slug** + extension.

- `health.jpg` → category with slug `health`
- `technology.png` → category with slug `technology`
- `food.webp` → category with slug `food`

Supported extensions (tried in order): `.jpg`, `.jpeg`, `.png`, `.webp`.

Example files: `health.jpg`, `technology.jpg`, `food.png`, etc.
